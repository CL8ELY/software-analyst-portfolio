Esta sección reúne los casos de estudio principales del portafolio. Cada caso está diseñado para mostrar cómo se abordó un problema desde análisis, cómo se estructuró la información del sistema y qué artefactos ayudaron a conectar necesidad, especificación y solución.

## Casos previstos

- Museo 26 de Julio
- Sistema de gestión apícola
- Sistema de gestión avícola

## Enfoque de publicación

Cada caso de estudio se construye a partir de documentación histórica real y se publica en una versión adaptada para uso profesional y público, respetando criterios de claridad, anonimización y confidencialidad.
